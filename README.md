# 🧮 Calculator

A simple calculator built with **HTML, CSS, and vanilla JavaScript**.

---

## ✨ Features

- **Basic arithmetic:** `+ - * /`
- **Percentage support:** `%`
- **Sign toggle:** `+/-`
- **Backspace and clear**
- **Decimal support**
- **Chain calculations**
- **Keyboard-free button UI**

---

## ⚙️ How it works

- Input builds a string expression  
- Expression is evaluated using `eval()`  
- Result is displayed after pressing `=`  
- Basic preprocessing handles percentages  

---

## 📁 Files

- `index.html` – UI structure  
- `style.css` – Styling  
- `scripts.js` – Calculator logic  

---

## ⚠️ Notes

- Uses `eval()` for simplicity (not production-safe)
- Floating point precision issues may occur
- No parentheses or scientific functions yet
